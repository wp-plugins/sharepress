Automatically send your Posts to your personal Facebook Wall. 

== Description ==

Sharepress automatically sends your WordPress Posts to your personal Facebook Wall. 

== Installation ==

1. Extract the zip archive into the plugin directory. This should create a folder named 'sharepress-lite'
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure Sharepress. Instructions in the application will help you to create your own Facebook Application (required)
